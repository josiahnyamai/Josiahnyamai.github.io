1. Indeed Remote Jobs Scraper

Using Python, Selenium, and Beautiful Soup,I extracted the job title, job link, job type, company name and job description for all remote jobs listed on the Indeed website in the following fields: Data Analyst,Data Engineer and Machine Learning Engineer. After scraping, I created a PostgreSQL database to store the data. I used Aiven to create a free PostgreSQL database.

2. Exploring the Kenyan Music Industry: Analyzing YouTube Data of Kenyan Musicians.

This project leverages the YouTube API to extract and analyze data on music videos from Kenyan musicians. By examining metrics such as view counts, likes, comments, and upload dates, the project aims to provide insights into the popularity trends, audience engagement, and growth patterns within the Kenyan music industry. The findings offer a comprehensive understanding of the digital presence and impact of Kenyan musicians on YouTube.

  • View On Github

3. CREDIT CARD FRAUD DETECTION MODEL.

I developed a credit card fraud detection model.I cleaned the data, then trained two machine learning models: Logistic Regression and Decision Tree. The Logistic Regression model proved to be the most accurate in identifying fraudulent transactions. This model can be applied in real-life scenarios: banking, e-commerce, payment gateways, and insurance companies to monitor transactions, detect and prevent fraud, ensuring the security of financial operations.
